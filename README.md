# 3.1C_SIT305_2026

This Android App, developed in Android Studio, allows users to complete a 5 question multiple-choice quiz. 
This app was developed based on provided mobile app wireframes, focusing on clean UI, and responsive interactions with the users.

Developed for SIT305 assessment by Thilini Fonseka

## Quiz App
This application is designed to provide an interactive quiz experience on Android devices. Users can answer multiple-choice questions, monitor their progress as they go, and receive a final score upon completion.

### Technology used
1. Java
2. Android Studio
3. XML

### App Features
1. Friendly and consistent UI design
2. Smooth navigation between pages
3. Real-time progress tracking
4. Remembers user name for future attempts

## How to run app
1. Clone the repository
   https://github.com/ThiliniPF/3.1C_SIT305_2026.git
2. Open the project in Android Studio
3. Allow Gradle to sync
4. Run the app using an emulator or a physical Android device
